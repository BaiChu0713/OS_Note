## Web前端开发基础概述

### HTML（超文本标记语言）
HTML 是构建网页内容的基础语言，通过各种标记（标签）来定义网页的结构和内容。它主要用于描述网页的文本、图像、链接、列表、表格等元素。HTML 文件以 `.html` 为后缀，浏览器通过解析 HTML 文件来渲染网页。

### CSS（层叠样式表）
CSS 用于控制 HTML 元素的样式和布局。它可以设置字体、颜色、间距、尺寸等属性，通过选择器来选择 HTML 元素，并为其应用样式规则。CSS 可以通过以下方式应用：

|类型|位置|
|--|--|
|**内联样式**|直接写在 HTML 标签的 `style` 属性中|
|**内部样式表**|写在 HTML 文件的 `<style>` 标签中|
| **外部样式表**|写在单独的 `.css` 文件中，并通过 `<link>` 标签引入 HTML 文件|

### JavaScript

JavaScript 是一种运行在浏览器中的脚本语言，用于为网页添加交互性。它可以操作 HTML 元素和 CSS 样式，响应用户的行为（如点击、输入等），并实现动态效果。JavaScript 代码可以写在 HTML 文件的 `<script>` 标签中，也可以写在单独的 `.js` 文件中，并通过 `<script>` 标签引入 HTML 文件。

## HTML

HTML是用于构建网页内容结构的基础语言。它以`.html`为后缀，浏览器通过解析HTML文件来渲染出用户看到的网页页面。

HTML的核心作用是**为网页提供结构化的信息**，使网页内容能够被浏览器正确解析和显示。它是一种标记语言，而不是编程语言，主要负责内容的组织和展示，而不涉及逻辑处理。

### （一）文本标记
- **标题**：`<h1>`到`<h6>`，用于定义不同级别的标题，`<h1>`是最高级别的标题，字体最大，`<h6>`是最低级别的标题，字体最小。
- **段落**：`<p>`，用于定义段落，浏览器会在段落前后自动添加一些间距。
- **换行**：`<br>`，用于在文本中强制换行，不包含结束标签。
- **加粗与斜体**：`<strong>`表示加粗文本，强调重要性；`<em>`表示斜体文本，用于强调语气。

### （二）图像与链接标记
- **图像**：`<img>`，用于插入图像，需要设置`src`属性来指定图像的路径，还可以通过`alt`属性为图像提供替代文本。
- **链接**：`<a>`，用于创建超链接，通过`href`属性指定链接的目标地址，可以链接到其他网页、文件或者网页内的锚点位置。

### （三）列表标记
- **无序列表**：`<ul>`和`<li>`，`<ul>`是列表的容器，`<li>`是列表项，列表项前通常会显示一个圆点等符号。
- **有序列表**：`<ol>`和`<li>`，与无序列表类似，但列表项前会显示数字编号。
- **定义列表**：`<dl>`、`<dt>`和`<dd>`，用于描述术语和定义，`<dt>`表示术语，`<dd>`表示定义。

### （四）表格标记
- **表格**：`<table>`，用于组织数据。
- **表格行**：`<tr>`，定义表格行。
- **表头单元格**：`<th>`，定义表格的表头单元格，通常会加粗并居中显示。
- **普通单元格**：`<td>`，定义表格的普通单元格。

### （五）表单标记
- **表单**：`<form>`，用于收集用户输入。
- **输入框**：`<input>`，有多种类型，如文本输入框（`type="text"`）、密码输入框（`type="password"`）、单选按钮（`type="radio"`）、复选框（`type="checkbox"`）等。
- **标签**：`<label>`，为表单控件提供标签，通过`for`属性与`<input>`的`id`属性关联，提高表单的可访问性。
- **按钮**：`<button>`，定义按钮，用于提交表单或者执行其他操作。
