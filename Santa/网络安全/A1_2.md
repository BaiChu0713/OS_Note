# web app
## 定义
web app运行于网络和标准浏览器上，基于网页开发技术实现特定功能的应用

|类型||
|-|-|
|前端|html，css，JavaScript|
|后端|java，python，php|
|数据库|mqsql，Oracle，mongodb|
|容器|window(IIS),Linux(Nginx,Apache)|
|协议|tcp,DNS,HTTP,HTTPS|

## 实际底层执行

1. 客户输入url——dns解析url得到ip地址——根据ip地址找出对应服务器
2. 客户机通过tcp/ip协议简历到web服务器的tcp链接
3. 客户机向web服务器发送http请求报文，请求服务器资源的资源文档
4. web服务器接收到http请求报文，向客户机发出http响应报文

|请求种类|操作|
|-|-|
|html文档|web服务器打开相应的html文档打开，将响应内容发送给给客户机|
|php文件|web服务器寻找、委托PHP应用服务器——php应用服务器解析成html静态代码发送给web服务器——后同HTML文档，大宋html静态代码给客户机|
|访问数据库|web服务器同构php应用服务器，进一步访问数据库|

客户机解析html静态文档，完成访问流程后客户机与服务器断开链接

| 序号 | Web服务器名称 | 描述 |
| --- | --- | --- |
| 1 | Apache HTTP Server（Apache） | 是Apache软件基金会的一个开源代码的网页服务器软件，可以在大多数电脑操作系统中运行，由于其跨平台和安全性被广泛使用，是最流行的Web服务器软件之一。 |
| 2 | Nginx | 是一个网页服务器，它能反向代理HTTP, HTTPS, SMTP, POP3, IMAP的协议链接，以及一个负载均衡器和一个HTTP缓存。Nginx是一款面向性能设计的HTTP服务器，相较于Apache、lighttpd具有占有**内存少，稳定性高**等优势。 |
| 3 | IIS | Internet Information Server的缩写。它是微软公司主推的服务器。 |
| 4 | Lighttpd | 是一个德国人领导的开源Web服务器软件,具有非常低的内存开销、CPU占用率低、效能好以及丰富的模块等特点。 |
| 5 | Tomcat | 是Apache软件基金会的Jakarta项目中的一个核心项目，由Apache、Sun和其他一些公司及个人共同开发而成。Tomcat技术先进、性能稳定，而且免费，因而深受Java爱好者的喜爱并得到了部分软件开发商的认可，成为目前比较流行的Web应用服务器。 |
