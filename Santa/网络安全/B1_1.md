# 环境配置——Kali



## VMware 与 Kali Linux 简介

### VMware
- **定义**：VMware 是一款“虚拟PC”软件，可在单台计算机上同时运行多个操作系统（如 Windows、DOS、Linux 等）。
- **运行方式**：与多启动系统不同，VMware 支持多个操作系统同时运行，像标准 Windows 应用程序一样切换。
- **功能特点**：
  - 虚拟分区和配置，不影响真实硬盘数据。
  - 支持虚拟机网络连接，可组建虚拟局域网。
- **适用场景**：适合学习和测试，但性能低于直接安装在硬盘上的系统。

### Kali Linux
- **定义**：Kali Linux 是基于 Debian 的 Linux 发行版，专为数字取证和渗透测试设计。
- **预装工具**：预装了多种渗透测试工具，如 nmap、Wireshark、John the Ripper 和 Aircrack-ng。
- **运行方式**：可通过硬盘、Live CD 或 Live USB 启动。
- **支持工具**：支持 Metasploit Framework，用于开发和执行针对远程主机的 Exploit 代码。
- **使用场景**：安装 VMware 后，可在虚拟机中启动 Kali Linux 进行渗透测试。

## Kali常见操作讲解
| 命令                | 描述                               | 示例                                 |
| :---------------- | :------------------------------- | :--------------------------------- |
| `pwd`             | 显示当前工作目录的路径                      | `pwd`                              |
| `ls`              | 列出当前目录中的文件和文件夹                   | `ls`                               |
| `cd`              | 切换目录                             | `cd /path/to/directory`            |
| `touch`           | 创建一个空文件                          | `touch demo.txt`                   |
| `cat`             | 查看文件内容                           | `cat demo.txt`                     |
| `echo`            | 输出字符串到终端或写入文件                    | `echo "Hello, World!" > demo.txt`  |
| `echo`            | 追加字符串到文件末尾                       | `echo "Hello, World!" >> demo.txt` |
| `rm -f`           | 强制删除文件，不提示确认                     | `rm -f demo.txt`                   |
| `mkdir`           | 创建文件夹                            | `mkdir tmp`                        |
| `rmdir`           | 删除空文件夹                           | `rmdir tmp`                        |
| `bzip2`           | 压缩文件为 `.bz2` 格式，不保留原文件           | `bzip2 demo.txt`                   |
| `bunzip2`         | 解压缩 `.bz2` 文件，不保留原文件             | `bunzip2 demo.txt.bz2`             |
| `gzip`            | 压缩文件为 `.gz` 格式，不保留原文件            | `gzip demo.txt`                    |
| `gunzip`          | 解压缩 `.gz` 文件，不保留原文件              | `gunzip demo.txt.gz`               |
| `tar -jcvf`       | 压缩文件为 `.tar.bz2` 格式              | `tar -jcvf demo.tar.bz2 demo.txt`  |
| `tar -tf`         | 查看 `.tar` 文件内容                   | `tar -tf demo.tar.bz2`             |
| `tar -zcvf`       | 压缩文件为 `.tar.gz` 格式               | `tar -zcvf demo.tar.gz demo.txt`   |
| `tar -zxvf`       | 解压缩 `.tar.gz` 文件                 | `tar -zxvf demo.tar.gz`            |
| `rar a`           | 使用 RAR 格式压缩文件（需安装 rar 工具），保留原文件  | `rar a package.rar package`        |
| `rar x`           | 解压缩 RAR 文件，保留原文件                 | `rar x package.rar`                |
| `apt-get install` | 安装软件包（适用于基于 Debian 的系统，如 Ubuntu） | `apt-get install x`                |
