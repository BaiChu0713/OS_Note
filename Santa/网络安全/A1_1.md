# 初始章节
## 前端开发工具
常见的有vsc(最流行)、atom(本人现在在使用的)、sublime text、vim

### 前端开发流程
> 前端开发就是创建web页面或app等前端界面呈现给用户的过程

涉及语言——html、css、js

## html

> 前后端走向

- 前端：页面表现、速度流畅、兼容性、用户体验
- 后端：高并发、高可用、高性能，安全，存储，业务

> 为什么要前后端分离？

1. 性能问题
2. 学习成本

### 介绍

组成：
  - 版本声明`<!DOCTYPE html>`
  - 头部`<head>`
  - 主体`<body>`

类型：
  - 标签`<p></p>`
  - 元素`<p>hi</p>`
  - 属性`<a href="xxx">`

## css

> 堆叠样式表——表现文件样式的计算机语言

css能对网页中元素位置的排版进行精确控制

不影响逻辑，只影响外观

## JavaScript

可插入html界面的动态脚本语言

它能引申出**xss攻击**、**文件上传漏洞**

Node.js就是运行在服务端的js，基于chrome JavaScript运行时建立的平台
